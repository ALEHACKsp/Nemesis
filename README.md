## User Interface
### Features 
- Simple and intuitive design
- Custom dump name
- Custom dump directory location
- Sortable process list
- Keyboard shortcuts

### Keyboard Shortcuts
<kbd>CTRL+D</kbd> - Dumps the selected process

<kbd>CTRL+R</kbd> - Refresh the processes

<kbd>CTRL+ALT+S</kbd> - Opens the settings
